# Eysenck Personality Questionnaire (EPQ)

Desktop application for psychological assessment based on the Eysenck Personality Questionnaire.

## Features
- Graphical user interface (Tkinter)
- Automated scoring
- Visualization of results

## Purpose
This project demonstrates implementation of psychological testing tools and basic data processing in Python.

## Technologies
- Python
- Tkinter

## Usage
Run:
python main.py
